Excel Macro & VBA

Berikut adalah topik-topik yang akan anda pelajari di dalam video tutorial ini:

  1. Intro – permulaan video je, nak bagitau apa yang kita akan belajar.
  2. Apa itu macro & bila patut guna – sini dia explain kenapa macro ni best bila kita nak automasi kerja.
  3. Cara rakam macro basic – tunjuk step by step macam mana nak mula rakam macro.
  4. Cara jalankan macro – lepas rakam, camne pulak nak run macro tu.
  5. Simpan workbook yang support macro – penting ni sebab bukan semua Excel file boleh simpan macro.
  6. Buka semula file macro – tunjuk cara nak access balik file yang ada macro tu.
  7. Beza Absolute & Relative references – penting bila rakam macro supaya action dia ikut betul-betul.
  8. Edit kod VBA untuk macro – kalau nak ubah atau upgrade macro, boleh edit kat VBA editor.
  9. Aktifkan Developer Tab – bar ni kena aktif dulu kalau nak buat macro.
  10. Guna Personal Macro Workbook – supaya macro boleh pakai kat mana-mana file.
  11. Tengok kod VBA masa rakam – boleh tengok live code yang tengah direkod masa buat macro.
  12. Guna If Statement – kalau ada condition, camne nak buat dalam macro.
  13. Guna For Loop – ulang sesuatu action ikut bilangan tertentu.
  14. Guna Input Box – untuk minta user masukkan data dalam macro.
  15. Guna Do While & Do Until Loop – nak ulang sesuatu sampai satu syarat dipenuhi.
  16. Gabungkan semua jadi 1 contoh – contoh besar yang satukan semua elemen tadi.
  17. Gabung banyak macro jadi satu – bila kita nak simplify, boleh combine.
  18. Tutup screen updating – untuk buat macro jalan laju tanpa ganggu paparan.
  19. Tiada butang undo lepas run macro – hati-hati sebab bila run macro, takleh undo.
  20. Penutup – rumus semua yang dah belajar dalam video ni.
