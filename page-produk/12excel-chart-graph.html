Berikut adalah topik-topik yang akan anda pelajari di dalam video tutorial ini:

  1. Intro – pembukaan video, explain secara ringkas apa itu Power BI dan apa yang akan dipelajari.
  2. Cara download Power BI Desktop – tunjuk langkah nak install software ni.
  3. Paparan mula (Welcome screen) – kenal dulu rupa interface bila buka Power BI.
  4. Ambil data masuk – tunjuk macam mana nak import data dari Excel, CSV atau sumber lain.
  5. Data view – bahagian ni untuk tengok data yang dah dimasukkan.
  6. Transform data guna Power Query – kalau data serabut, boleh kemas dan susun kat sini.
  7. Model view – setting hubungan antara jadual data (relationship view).
  8. Bina laporan dengan visual – buat grafik, chart dan visual menarik dari data.
  9. Tambah pages – boleh tambah beberapa page untuk buat dashboard lebih lengkap.
  10. Publish laporan – cara nak share atau upload report ke Power BI service.
  11. Penutup – ringkasan semua step dan kesimpulan akhir.
