Berikut adalah topik-topik yang akan anda pelajari di dalam video tutorial ini:

  1. Mula buat dashboard dinamik – dia tunjuk cara bagi dashboard kita auto-update bila data berubah.
  2. Guna named ranges atau table – untuk mudahkan kita link data dengan chart.
  3. Guna dropdown atau slicer – supaya user boleh pilih data ikut kategori atau tarikh.
  4. Rangka struktur dashboard – susun ruang dashboard ikut fungsi dan tujuan.
  5. Buat carta dan visual – mula create chart seperti bar, line atau pie ikut jenis data.
  6. Letak elemen tambahan – contohnya label, percentage, legend supaya info lebih jelas.
  7. Ubah style visual – tukar warna, font, border dan position ikut tema yang kita nak.
  8. Gabungkan semua dan kemaskan – finalize dashboard dengan susunan yang kemas dan professional.
